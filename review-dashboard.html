Semantic colours ✅ Good

Consider documenting these as official semantic tokens in Figma so they're consistent across tools.
Typography ⚠️ Partial
File uses Segoe UI Bold / Semibold. Brand guidelines specify Noto Sans Regular (400). Segoe UI is a reasonable Power BI fallback — Noto Sans is not a built-in system font in PBI.
Document Segoe UI as the official Power BI font exception. Confirm with Pablo whether Noto Sans can be embedded or loaded via a custom visual.
Minimum font size ⚠️ Minor
Data label size is set to 9pt. BSI brand minimum body text is 10pt. Title (10pt) and header (11pt) are compliant.
Bump label fontSize from 9 → 10 in the JSON, or document 9pt as an accepted exception for dense data labels.
Extended colours (9–32) ⚠️ Minor
Slots 9–32 use 24 non-brand colours for charts with 9+ data series. Most dashboards won't reach this limit. The methodology for how these were chosen is undocumented.
Key meeting question: ask Pablo how the extended palette was constructed. If systematic, capture it in Figma variables. If ad-hoc, consider generating a brand-derived extended set.
